# GrometZappIA Client (Frontend)

Frontend do GrometZappIA construído com Next.js 14.

## ✅ Requisitos

- Node.js 18+
- Backend rodando (API)

## ⚙️ Configuração

Crie/ajuste o arquivo .env com a URL da API:

```
NEXT_PUBLIC_API_URL=http://localhost:5001/api
```

## ▶️ Executar em desenvolvimento

```
npm install
npm run dev
```

O servidor de desenvolvimento inicia em http://localhost:3000.
